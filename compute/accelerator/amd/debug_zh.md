# debug - 中文翻译

## 工具

### ROCR_VISIBLE_DEVICES

选择特定的GPU（相当于`CUDA_VISIBLE_DEVICES`）：

```bash
ROCR_VISIBLE_DEVICES=0,1 python my-program.py
```

一个方便的别名用于实时查看更新：
```bash
alias wr='watch -n 1 rocm-smi'
```

这个命令同时显示CPU和GPU的信息。
